配置过程

动态壁纸

设置动态壁纸，我用的是hidamari

软件安装

注意:使用该方法安装的话软件下载速度在国内可能会很慢

有条件的可以上代理或者VPN(最好是VPN)

安装和配置Flatpak

sudo apt update
sudo apt install flatpak
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
sudo reboot

重启完成后，就可以安装hidamari了

flatpak install flathub io.github.jeffshee.Hidamari

一般来说，你就够在应用列表里找到这个软件

如果不能，你可以这样运行

flatpak run io.github.jeffshee.Hidamari

桌面小组件

我使用的是conky

软件安装

kali可以直接通过apt进行安装

sudo apt update
sudo apt install conky-all

官方有很多各位大神做好的配置

创建/修改配置文件

注意:配置文件放在~/.config/conky

并命名为conky.conf

cd ~/.config/conky
vi conky.conf

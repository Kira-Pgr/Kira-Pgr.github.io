如何拯救IT人的英语

在进入IT圈后，我发现一个事情…

很多IT人学英语有困难

作为一个英语学的还不错的猫猫，

我准备用这篇文章给大家推荐一个不同寻常的英语学习方法

一条小众的英语学习道路

Tip: 每个人都有最合适自己的学习方式,

所以本篇文章只是简单讲解一下学习思路和提供一些指引~~

真正适合的学习方式还是要各位自己探索喵

希望这篇文章能帮到大家吧.QwQ.

Stage 1: 摆脱英语小白状态

Tip 1: 不要害怕学英语

很多IT人一直因为自己英语不好就对学英语产生了畏难情绪

实际上这个是duck不必的

有时候越怕反而更加没办法学好了

要首先从精神上战胜嘤glish(

各位可以把英语当成一个辅助各位学习自己擅长东西的工具来看

(而不是一个特别难的科目)

Q: 我学英语没天赋啊啊啊啊

A: 爪巴吧,很可能别人只是比你多学了几年，从小开始学的那种= =

Tip 2: 脱离对翻译软件的依赖

对于IT来讲，翻译过的专业文本会看上去很难受= =,

而且用翻译软件也没办法提升各位的英文水平

所以推荐各位在看英文的东西的时候尽量不用翻译软件, 或者只翻译个别词

时间长了，各位就会发现看原文比看机翻过后的东西舒服了

Tip 3: 整体思想!!!

一个英文句子应该是当做一个整体或者几个小整体来看

但是捏，很多人学着学着应试英语就终于学废了!!!

光背单词，记语法, 到头来就只会一个一个单词地翻译,句子直接看不懂

出现这种问题的人可以试试忍住不翻译单词，等看完一部分句子或者一整句再去理解意思

Tip 4: 直接阅读英文文本

很多人讲的要用英文思维阅读英文，实际上就是这一点:)

各位可以尝试在阅读英文文本的时候不要翻译成中文之后再理解，

直接在阅读英文的过程中理解

Tip 5: 有意无意积累单词

讲真，让我连续好久背一堆单词我坚持不下去(

所以我就把这个积累单词的任务放在了平时

虽说是任务，但很多时候不经意间就完成了

比如说，有些人比较喜欢看美剧，听英文歌

歌词或者台词里面就经常发现有些有趣的单词

这个时候你花很短时间就记住了awa

比如说:

我最近用了一个C2工具，名字叫Havoc

觉得很有趣然后就查了下这个单词的含义: 毁灭

Tip 6: 沉浸式学英语

各位可以尝试给自己一个纯英文的环境

比如说把系统/软件语言换成英文,用谷歌查资料等等

这样英文水平提升的速度会比较快捏~

当然如果你直接去国外的话,,,算是最好的方法

Tip 7: 注意语音语调

有些人说的英文就听起来特别"难受"

那就是因为有那部分人在学习英语的时候没注意语音语调

可以通过模仿一些外国人的英语来让各位的语音语调听起来很舒服

(当然，最好不要模仿印度英语/非洲英语)

Stage 2: 逐渐提升

当各位已经有一定英语基础的时候，就可以尝试以下下面两个小建议说的方法了:>

Tip 8: 看英文释义

在有一定的词汇量的时候，就可以尝试通过英文释义认识新的词汇

虽说可能英文释义中会有一大堆各位可能更加不懂的单词

但这样做会让你的英语水平加速提升的哦qwq

Tip 9: 学会猜测词义

我有过这样的经历

虽说同学A的词汇量比我大，但同学A看不懂的文章我能看懂

这样事情发生的原因就是我会依据上下文猜测生词/专业术语的含义

这种能力在IT专业英语中可以说是非常重要啦

猜测生词

一个句子里有两三个生词其实问题不是很大，你可以看看上下文意思来推测出那些生词的意思

即使猜不出来，某些生词不看它们都不影响你整体的理解

猜测专业术语

一般专业术语基本上都是名词(IT术语也是一样)

名词: 一种东西

所以看到一个专业术语的时候，各位就可以暂且把它理解为"某个东西"，在文章中这个术语可能会反复出现，然后就可以依据文中的描述推测出"这个东西"是"干什么用的",或者"为什么会有这个东西".

然后就可以依据各位已有的专业知识来往中文的一些术语上靠，最贴切的中文术语往往就是那个英文术语的译文.

(实际上这个时候各位已经理解了这个术语的含义，其实已经没必要做最后一步的翻译工作了)

Stage 3: 让英语成为促进你学习的工具

让英语帮助各位学技术吧! Nya~

Tip 10: 去各种国外优秀技术网站

  • StackOverflow

  • GitHub

  • StackExchange

  • …

Tip 11: Google一下，你就知道

在Google用英文查资料，真的很帮助各位解决IT技术问题哦~~

Stage 4: This is not the end

做到这里后，想必英语已经不是各位学习道路上的"拦路虎"了吧

接下来要做的就是按照自己之前的学习方式继续探索了哦qwq

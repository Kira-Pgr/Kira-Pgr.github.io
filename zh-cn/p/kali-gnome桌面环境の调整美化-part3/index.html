主题安装

一般主题会在页面中写上详细的安装方式，我就在这里简述一下吧.qwq.

下载主题包

  • 第一种方案:下载到/usr/share/themes/

    # 如果没有这个目录创建一下
sudo mkdir /usr/share/themes/
cd /usr/share/themes/
sudo git clone 主题GitHub链接 主题名

  • 第二种方案:下载到~/.themes/

    # 依旧是没有目录就创建一下
mkdir ~/.themes/
cd ~/.themes/
git clone 主题GitHub链接 主题名

应用主题

直接上命令吧，最近总想摆大烂

gsettings set org.gnome.desktop.interface gtk-theme "主题名"
gsettings set org.gnome.desktop.wm.preferences theme "主题名"
